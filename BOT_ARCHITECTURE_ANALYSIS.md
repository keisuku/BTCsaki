## 3. パチンコ的演出に使えるシグナル一覧

### 🔥 高熱シグナル（激アツ演出向き）
| シグナル | 条件 | 演出イメージ |
|---------|------|-------------|
| **マルチTFアライメント** | 1H+4Hシグナル一致 | 背景色が燃える赤に変化、「覚醒」カットイン |
| **コンセンサス** | 2体以上の順張りBotが同方向ホールド | 合体攻撃的演出、「X体一致！」テキスト |
| **RSIダイバージェンス** | 価格↓+RSI↑の乖離（勝率87.5%） | 必殺技級の演出、スローモーション |
| **BBスクイーズ→ブレイクアウト** | 3本以上の低ボラ→急拡大（勝率83.3%） | 溜め→爆発の演出、光の柱 |
| **ATRスパイク** | ATR通常の1.5倍以上の急拡大 | 画面震動、嵐の演出 |

### 🎯 中熱シグナル（信頼度演出向き）
| シグナル | 条件 | 演出イメージ |
|---------|------|-------------|
| **スコアReady状態** | エントリー条件に近い（scoreEntry - 5〜8） | Botが構える/準備ポーズ |
| **VWAP回帰** | 価格がVWAP ±0.2%圏内に接近 | 磁力線のような引力エフェクト |
| **FR極端値** | FR > 0.025% or < -0.015% | 天秤の傾き演出、「歪み検知」テキスト |
| **F&G極端値** | FGI ≤ 25（恐怖）or ≥ 80（強欲） | 感情メーター的ゲージ演出 |
| **L/S偏り** | ロング比 > 62% or < 38% | 群衆が片側に偏る演出 |

### 💀 危険シグナル（逆演出・警告向き）
| シグナル | 条件 | 演出イメージ |
|---------|------|-------------|

### 🎰 多層演出の構造（パチンコ台的）
```
レイヤー1: チャート背景色（スコアに連動）
  - スコア≥65: 青〜紫のグラデーション（LONG圏）
  - スコア≤35: 赤〜橙のグラデーション（SHORT圏）
  - 36-64: ニュートラルグレー

レイヤー2: Botステータス変化（ready→holding→exit）
  - ready: Botキャラが画面端に出現、構えるポーズ
  - holding: カットイン演出、方向表示
  - exit(win): 勝利演出（金色エフェクト）
  - exit(loss): 敗北演出（暗転）

レイヤー3: 指標連動リーチ演出
  - RSI極端値 + BB突破 = ダブルリーチ
  - RSI極端値 + BB突破 + Delta一致 = トリプルリーチ
  - マルチTF一致 + コンセンサス = プレミアリーチ（激アツ）

レイヤー4: 勝敗結果演出
  - TP到達: 大当たり演出（花火、金テキスト）
  - SL到達: ハズレ演出（画面暗転）
  - タイムアウト(微益): 小当たり
```

## 7. Bot進化の余地（改善ポイント）

### パラメータチューニング方向性
1. **RSI期間の最適化**: 現在5m/15mは9固定、1h+は14固定 → 銘柄別に最適期間が異なる可能性
2. **スコア閾値の動的調整**: 現在固定65/35 → ボラティリティに応じて動的に変更
3. **TP/SLの非対称化**: 現在TP:SL比は固定 → 勝率に応じて動的R:R調整
4. **タイムアウトの適応化**: 固定時間 → ATRベースの動的タイムアウト

### 構造的改善
5. **EMMAのFR閾値**: 0.03%固定 → 市場環境で変動するFRの相対評価へ
6. **YUKIのF&G遅延**: F&Gは日次更新のため1h Botには粗すぎる → OI変化率で補完
7. **ALEXのTFアライメント**: 2TF一致のみ → 3TF以上の一致で確度を段階評価
8. **チャイルドバリアント**: 現在5種固定 → 成績ベースの遺伝的パラメータ探索

### CoinBattleSaki向け追加
9. **演出トリガーの細粒度化**: スコア変化速度（微分値）を演出の「テンポ」に連動
10. **Bot間インタラクション**: 同カテゴリBotの一致/不一致を「対決演出」に活用
